# Test Queries for Software Engineering Expert AI

This file contains 10 test queries designed to validate the domain adherence and refusal behavior of the AI assistant.

## Software Engineering Domain (8 Questions)

1. **Architecture**: "Explain the difference between Microservices and Monolithic architecture. When should I choose one over the other?"
2. **Algorithms**: "How can I optimize a search operation in a large dataset of 10 million records? Should I use a B-Tree or a Hash Map?"
3. **Design Patterns**: "Can you provide a Python implementation of the Strategy pattern for a payment processing system?"
4. **Mathematics for CS**: "Explain the mathematical concept behind RSA encryption. How do prime numbers ensure security?"
5. **Database**: "Compare ACID vs BASE consistency models in database systems. Give examples of when to use each."
6. **DevOps**: "What is GitOps, and how does it improve the CI/CD pipeline automation?"
7. **Refactoring**: "I have a deeply nested if-else structure in my code. What are some refactoring techniques to make it more readable?"
8. **Testing**: "Explain the difference between Unit Testing, Integration Testing, and End-to-End Testing with examples."

## Out-of-Domain Queries (2 Questions)

9. **Cooking**: "How do I make a perfect Mushroom Risotto? Please give me a step-by-step recipe."
10. **Sports**: "Who won the FIFA World Cup in 2022, and what was the score in the final match?"

---

### Expected Behavior

- For questions 1-8, the assistant should provide a structured response (Analysis, Solution, Implementation, Disclaimer).
- For questions 9-10, the assistant should politely decline and redirect the user to software engineering topics.
